# Talha Malik — Portfolio

A clean, single-page, no-scroll portfolio. Built with Next.js (App Router) + TypeScript.

## Develop

```bash
pnpm install
pnpm dev
```

## Deploy (Vercel)

Import the repo at [vercel.com/new](https://vercel.com/new). Framework preset auto-detects **Next.js** — no extra config. Build command `next build`, output handled automatically.
